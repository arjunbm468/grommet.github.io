Anchor

A text link. We have a separate component from the browser base so we can style it. You can either set the icon and/or label properties or just use children.

Child textLabelEdit

Options

disabled true|false
Whether to disable the anchor. Defaults to false.
href {string}
Hyperlink reference to place in the anchor.
icon {element}
Icon element to place in the anchor. See Icon.
label {string}
Label text to place in the anchor.
onClick {function}
Click handler.
primary true|false
Whether this is a primary anchor.
reverse true|false
Whether an icon and label should be reversed so that the icon is at the end of the anchor.
tag {string}
The DOM tag to use for the element. The default is <a>. This should be used in conjunction with components like Link from React Router. In this case, Link controls the navigation while Anchor controls the styling.
target _blank|_self|_parent|_top|{string}
Target of the link.

Examples

Default

Child text
<Anchor href="" onClick={...}>
  Child text
</Anchor>

Primary, Label

Label
<Anchor href="" label="Label" primary={true} onClick={...} />

Icon, Label

Edit
<Anchor href="" icon={<Edit id="edit-icon-1" />} label="Edit" onClick={...} />

Icon, Label, Reverse

Next
<Anchor href="" icon={<Next />} label="Next" reverse={true} onClick={...} />

Icon

<Anchor href="" icon={<Edit id="edit-icon-2" />} onClick={...} />

Target

Label
<Anchor href="" target="_blank" label="Label" onClick={...} />

Disabled

Text
<Anchor primary={true} disabled={true}>
  Text
</Anchor>
