Notification

A box to display Notification messages.

<Notification message="..." />

Options

message {string}
Message to display in the notification box.
percentComplete {number}
Number to measure the progress of an ongoing notification.
size small|medium|large
Size of the notification box.
state {string}
State of the element on the notification message (e.g. Active, Running).
status {string}
Status of the element on the nofitication message (e.g. Ok, Critical).
timestamp {date}
Timestamp of the notification message.

Options for Box are also available for Notification.

Examples

Unknown

Unknown Message
<Notification message="Unknown Message" />

Warning

Inconsistent configuration detected.
Mon Jan 25 2016 00:00:00 GMT+0000 (UTC)
Active
<Notification status="warning" message="Inconsistent configuration detected." timestamp={"2016-01-25T00:00:00.000Z"} state="Active" />

Critical

Temperature threshold exceeded by 10 degrees.
Mon Jan 25 2016 00:00:00 GMT+0000 (UTC)
Active
<Notification status="critical" message="Temperature threshold exceeded by 10 degrees." timestamp={"2016-01-25T00:00:00.000Z"} state="Active" />

OK

Updated server configuration.
Mon Jan 25 2016 00:00:00 GMT+0000 (UTC)
Completed
<Notification status="ok" message="Updated server configuration." timestamp={"2016-01-25T00:00:00.000Z"} state="Completed" />

Percent Complete

Updating server profile.
Mon Jan 25 2016 00:00:00 GMT+0000 (UTC)
Running
30%
<Notification status="ok" message="Updating server profile." timestamp={"2016-01-25T00:00:00.000Z"} state="Running" percentComplete={30} />
