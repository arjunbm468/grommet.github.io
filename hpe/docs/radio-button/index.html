RadioButton

A radio button in a web form. We have a separate component from the browser base so we can style it.

<RadioButton id="item2" label="Item 1">

Options

checked true|false
Same as React <input checked= >.
defaultChecked true|false
Same as React <input defaultChecked= >.
disabled true|false
Same as React <input disabled= >.
id {text}
The DOM id attribute value to use for the underlying<input> element.
label {text|node}
Label text to place next to the control.
name {text}
The DOM name attribute value to use for the underlying<input> element.
onChange {func}
Same as React <input onChange= >.

Example

Basic

<div>
  <RadioButton id="choice1-1" name="choice1" label="Choice 1" checked={true} onChange={...} />
  <RadioButton id="choice1-2" name="choice1" label="Choice 2" checked={false} onChange={...} />
</div>

Disabled

<div>
  <RadioButton id="choice2-1" name="choice2" label="Choice 1" checked={true} disabled={true} onChange={...} />
  <RadioButton id="choice2-2" name="choice2" label="Choice 2" checked={false} disabled={true} onChange={...} />
</div>
