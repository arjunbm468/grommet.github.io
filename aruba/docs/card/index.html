Card

The Card module.

Card Options

thumbnail {url}
Url path to image. Use reverse option to position thumbnail within card.
label {string}
Content label.
heading {string}
Content heading.
description {string}
Content description.
link {element}
Anchor element to place in the anchor. See Anchor.
onClick {function}
Click handler.
video {source: <string>, type: mp4|webm|ogg}|{element}
Video media type and source path. Also accepts a Video element.
direction column|row
Applies the Card in a column (default) or row direction. Expects multiple Card modules to be wrapped in a Tiles component.
reverse true|false
If thumbnail url is set, align thumbnail to top or bottom of card. Defaults to false.

Options for Tile are also available for Card.
